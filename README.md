# Øyna AI System

Dette repoet inneholder modeller og dokumentasjon for AI-agenten og den digitale tvillingen til Øyna Vassverk.

- `models/v1` – JSON-modeller (Master System Model, Digital Twin, Knowledge Graph, API Contract, Manifest)
- `docs` – Teknisk dokumentasjon
- `tools` – Fremtidige verktøy for validering, analyser og embeddings
